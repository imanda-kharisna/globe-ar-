# Globe AR Web App

Nama  : Imanda Kharisna
NIM   : 201220076
Mata Kuliah : AR / VR

## Deskripsi
Aplikasi AR berbasis Web menggunakan p5.js + p5.xr.
Pengguna dapat menempatkan Globe 3D di dunia nyata dengan Tap to Place.

## Fitur
- Tap to Place Globe AR
- Audio SFX saat tap
- Tekstur bumi realistis
- Satellite orbit animasi
- WebXR (fallback WebGL)

## Teknologi
- p5.js
- p5.xr
- WebXR
- GitHub Pages
